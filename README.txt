How to correctly compile and run the "shell":

 1 - Open terminal in current directory;
 2 - Make sure 'gcc' is installed running "gcc --version";
 3 - Compile using the command "gcc *.c -o shell -lm -lreadline -Wall -Werror";
 4 - Execute shell with "./shell":
 5 - Have fun!
